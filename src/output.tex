Znamenskaya forbade doing this, but: $$ ({ { sin  {  x  }  }  ^  {  2  } })' = { {  {  2  }  \cdot  {  { sin  {  x  }  }  ^  {  {  2  }  -  {  1  }  }  }  }  \cdot  {  { cos  {  x  }  }  \cdot  {  1  }  } } $$
